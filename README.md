# Image Loader

WPF-приложение для загрузки изображений из интернета по URL.

![demo](demo.gif)

## Функциональность

- Одновременная загрузка трёх изображений по отдельным URL
- Независимая остановка и перезапуск каждой загрузки
- Кнопка запуска всех загрузок сразу
- Прогресс-бар с количеством активных загрузок
- Симулятор качества сети (Low / Medium / High / Excellent)

## Технологии

- .NET 10, WPF
- MVVM (CommunityToolkit.Mvvm)
- Async/await, CancellationToken
